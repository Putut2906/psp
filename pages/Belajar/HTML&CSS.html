Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup untuk membuat halaman web. HTML menggunakan tag untuk menandai elemen-elemen dalam halaman web.

Struktur Daftar

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Belajar HTML</title>
</head>
<body>
    <h1>Halo, Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
</body>
</html>

Elemen Dasar HTML

Tag Fungsi
<h1> sampai <h6> Heading (judul) dari besar ke kecil
<p> Paragraf
<a href="url"> Link (tautan)
<img src="gambar.jpg" alt="deskripsi"> Gambar
<ul> / <ol> / <li> Daftar (bullet & numbering)
<table> / <tr> / <td> Tabel
<form> Formulir input pengguna

Contoh

<h2>Judul Subbab</h2>
<p>Ini adalah paragraf dengan <strong>teks tebal</strong> dan <em>teks miring</em>.</p>
<a href="https://www.google.com">Klik di sini untuk ke Google</a>
<img src="gambar.jpg" alt="Deskripsi Gambar">

List dalam HTML

  1. Unordered List (Daftar Tidak Berurut)
<ul>
    <li>Apel</li>
    <li>Pisang</li>
    <li>Jeruk</li>
</ul>
  1. . Ordered List (Daftar Berurut)
<ol>
    <li>Apel</li>
    <li>Pisang</li>
    <li>Jeruk</li>
</ol>

Gambar & Link dalam HTML

  1. Menampilkan Gambar

<img src="https://via.placeholder.com/150" alt="Gambar Contoh">

  1. Membuat Link

<a href="https://www.example.com" target="_blank">Kunjungi Situs</a>

Formulir HTML

        <form action="proses.php" method="post">
    <label for="nama">Nama:</label>
    <input type="text" id="nama" name="nama"><br>

    <label for="pesan">Pesan:</label>

</form>

Layout Sederhana

HTML Code 
<!DOCTYPE html>
    <html lang="id">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Website Sederhana</title>
        <style>
            body { font-family: Arial, sans-serif; }
            .container { width: 80%; margin: auto; }
            header, footer { background: #333; color: white; text-align: center; padding: 10px; }
            nav { background: #666; padding: 10px; }
            nav a { color: white; margin: 10px; text-decoration: none; }
            .content { padding: 20px; }
        </style>
    </head>
    <body>
    
        <header>
            <h1>Selamat Datang</h1>
        </header>
    
        <nav>
            <a href="#">Home</a>
            <a href="#">Tentang</a>
            <a href="#">Kontak</a>
        </nav>
    
        <div class="container">
            <div class="content">
                <h2>Judul Artikel</h2>
                <p>Ini adalah contoh artikel sederhana.</p>
            </div>
        </div>
    
        <footer>
            <p>© 2025 Belajar HTML</p>
        </footer>
    
    </body>
    </html>
