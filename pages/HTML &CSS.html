Pengertian HTML

HTML, singkatan dari HyperText Markup Language, adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML bukan bahasa pemrograman karena tidak memiliki logika pemrograman seperti variabel atau fungsi, melainkan merupakan bahasa markup yang digunakan untuk mengatur struktur dan elemen-elemen dalam dokumen web.

HTML digunakan untuk mendefinisikan struktur konten pada halaman web. Setiap elemen HTML ditulis menggunakan tag, yang biasanya memiliki bentuk seperti ini: .... Tag terdiri dari tanda kurung sudut (<..>) dan nama elemen, serta biasanya diikuti oleh konten atau atribut tertentu.

Secara keseluruhan, fungsi HTML adalah membangun struktur dasar dari halaman web, menampilkan konten, dan memungkinkan integrasi dengan teknologi lainnya. Tanpa HTML, halaman web modern tidak dapat dibuat atau ditampilkan di browser. Sebagai bahasa markup standar, HTML adalah fondasi untuk pengembangan web yang lebih lanjut.

Struktur Dasar HTML

Struktur dasar HTML adalah rangkaian elemen yang membentuk halaman web. Setiap elemen memiliki tujuan tertentu dan ditulis dalam format tag. Berikut adalah penjelasan lengkap tentang struktur dasar HTML:

<!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <meta name="description" content="Deskripsi halaman di sini">
                <meta name="keywords" content="kata, kunci, di, sini">
                <meta name="author" content="Nama Penulis">
                <link rel="stylesheet" href="styles.css">
                <title>Judul Halaman</title>
            </head>
            <body>
                <header>
                    <h1>Judul Utama</h1>
                    <nav>
                        <ul>
                            <li><a href="#section1">Section 1</a></li>
                            <li><a href="#section2">Section 2</a></li>
                            <li><a href="#section3">Section 3</a></li>
                        </ul>
                    </nav>
                </header>
            
                <main>
                    <section id="section1">
                        <h2>Section 1</h2>
                        <p>Isi dari section 1.</p>
                    </section>
                    <section id="section2">
                        <h2>Section 2</h2>
                        <p>Isi dari section 2.</p>
                    </section>
                    <section id="section3">
                        <h2>Section 3</h2>
                        <p>Isi dari section 3.</p>
                    </section>
                </main>
            
                <footer>
                    <p>© 2025 Nama Anda. All rights reserved.</p>
                </footer>
                
                <script src="script.js"></script>
            </body>
            </html>
  • <!DOCTYPE html>Ini adalah deklarasi tipe dokumen yang memberi tahu browser bahwa dokumen ini adalah HTML5.
  • <html> Elemen ini merupakan elemen root dari sebuah halaman web. Semua elemen HTML lainnya akan berada di dalam elemen <html>
  • <head> Bagian ini berisi informasi metadata tentang halaman, seperti judul halaman, penghubung ke file CSS, skrip JavaScript, dan informasi lainnya yang tidak ditampilkan langsung di halaman web. Beberapa tag dalam bagian <head> meliputi:
    1. <title> Menentukan judul halaman yang muncul di tab browser.
    2. <meta> yaitu Memberikan informasi tentang karakter encoding atau deskripsi halaman. <meta> terdiri :
      • <meta charset="UTF-8"> : Mengatur encoding karakter (UTF-8 mendukung semua bahasa).
      • <meta name="viewport" content="width=device-width, initial-scale=1.0"> : Mengatur responsivitas pada perangkat yang berbeda.
      • <meta name="description" content="..."> : Menyediakan deskripsi halaman untuk SEO.
    3. <link> Menghubungkan file eksternal, seperti file CSS.
    4. <script> Menyisipkan kode JavaScript
  • <body> Bagian ini berisi konten yang akan ditampilkan di halaman web. Semua elemen yang terlihat oleh pengguna berada di dalam elemen <body>, seperti teks, gambar, tabel, dan formulir. Struktur Konten dalam <body> ada berbagai elemen yang dapat digunakan untuk menyusun konten halaman.

Elemen Heading dan Teks

    Beberapa elemen dasar yang sering digunakan
  1. <h1> hingga <h6> : Untuk membuat heading (judul), dari yang paling penting <h1> hingga <h6>.
  2. <p> Digunakan untuk membuat paragraf.
  3. <br> : Menambahkan baris baru di dalam teks.
  4. <hr> : Menyisipkan garis horizontal sebagai pemisah.
  5. <strong> : untuk teks yang penting (dicetak tebal)
  6. <em> : untuk teks yang diberi penekanan (dicetak miring).

Elemen Link dan Gambar

  1. <a> : Membuat tautan ke halaman lain atau lokasi tertentu. Contoh:
    <a href="https://example.com">Tautan<a/>
  2. <img> : Menyisipkan gambar. Contoh :
    <img src="image.jpg" alt="Deskripsi Gambar">

    • Elemen Struktur Halaman

    • <header> : Bagian atas halaman, seperti logo atau menu navigasi.

    • <aside> : Konten tambahan, seperti sidebar atau catatan.
    • <footer> : Bagian bawah halaman, seperti hak cipta atau informasi kontak.
