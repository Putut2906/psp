Panduan Elemen HTML

Elemen Daftar

HTML 
                <ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>
  • <ol> Membuat daftar berurutan (numbered list).
  • <ul> Membuat daftar tidak berurutan (bullet list).
  • <li> Elemen untuk setiap item dalam daftar.

Elemen Formulir

  • <form> Membuat formulir untuk input pengguna.
  • <input> Elemen input (teks, tombol, dll).
HTML 
                <input type="text" placeholder="Nama Anda">
  • <textarea> Membuat area teks multi-baris.
  • <button> Membuat tombol.
  • <label> Memberikan label untuk elemen input.

Elemen Multimedia

  • <audio> Menyisipkan audio.
HTML 
                <audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>
  • <video> Menyisipkan video.
HTML 
                <video controls>
    <source src="video.mp4" type="video/mp4">
</video>

Elemen Khusus

  • <table> Membuat tabel data.
Judul Kolom
Data
  • <div> Elemen pembungkus serbaguna untuk mengelompokkan konten.
  • <span> Membungkus teks untuk styling tertentu.
